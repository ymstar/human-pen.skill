# 小红书创作指南

> 适配版本：v1.1 | 更新日期：2025-06

## 一、小红书的AI检测机制

### 1.1 官方AI治理立场

小红书于2025年4月发布AI治理主张，明确"鼓励+禁止"双清单：

**鼓励方向**：
- AI视觉创作（具有风格辨识度的艺术插画、叙事逻辑清晰的视频）
- AI角色创作（原创虚拟角色、有亮点的IP衍生）
- AI知识科普（复杂知识可视化、有信息增量的教程）

**禁止行为**：
- AI违规运营（自动托管账号、批量洗稿）
- AI造假（克隆声音面孔、虚假人设种草、伪造经历）
- AI侵权（未经授权生成可识别肖像、模仿版权作品）
- AI低质（套用模板批量生产、猎奇博眼球）

**处罚措施**：限制曝光 → 警告 → 封禁（梯度处理）

### 1.2 AI内容识别机制

平台从两个维度识别AI内容：

| 识别方式 | 说明 |
|---------|------|
| 用户主动标识 | 创作者发布时勾选"AI合成内容"声明 |
| 平台技术识别 | 图片/视频 AI生成痕迹检测、文本风格分析、账号行为建模 |

**关键数据**：
- 纯AI生成内容识别准确率达98.6%
- 人工润色内容识别准确率约87%
- 2025年以来已处置AI不良行为超百万例，其中AI托管账号超80万个

### 1.3 去AI痕迹的核心策略

1. **主动标识AI参与度**：即使内容经过AI润色也建议标注
2. **保持个人风格**：固定的人设标签、口癖、表达习惯
3. **增加真实细节**：具体时间、地点、人物、数字
4. **保留不完美**：允许打字错误、口语化表达
5. **增加互动痕迹**：评论区回复粉丝问题

## 二、小红书内容特点

### 2.1 笔记体特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 第一人称 | 真实分享感 | "我用了3个月..." |
| 场景化 | 代入感强 | "早八通勤"、"出差带这些" |
| 问题导向 | 解决具体问题 | "油痘肌怎么选粉底" |
| 结果导向 | 展示实际效果 | "持妆8小时实测" |

### 2.2 Emoji使用规范

- 每篇笔记使用5-15个emoji
- 分隔内容用emoji替代标点
- 重点标注用emoji突出

**高频emoji组合**：
```
✅ ❌ ⚠️ 🔥 💯 📌 💡 🙋‍♀️ 🤔 😭 哈哈哈
```

### 2.3 标题党特征

**禁用词汇**（触发限流）：
- 极限词：最、第一、必买、全网最强
- 诱导词：不转不是中国人、震惊、绝了

**推荐结构**：
```
[人群/场景] + [核心痛点/利益点] + [数字/效果] + [情绪词]
```

**爆款标题公式**：
```
公式1：人群+痛点+解决方案
→ "油痘肌早八人！持妆8小时不卡粉，亲测3款百元粉底液"

公式2：数字+效果+感叹
→ "3步搞定！素颜出门被夸皮肤好"

公式3：对比+悬念
→ "月薪3千和3万的穿搭差距在哪？"
```

## 三、去AI痕迹的小红书特有策略

### 3.1 语言风格

**避免**：
- 书面化表达
- 完美无缺的描述
- 过于系统的总结

**推荐**：
```
AI风："这款产品具有优异的保湿效果，能够有效改善肌肤状态"
人写："用了快一个月，最明显的感觉是上妆不卡粉了"

AI风："综上所述，该产品具有较高的性价比"
人写："百元内能买到这个效果，还要啥自行车"

AI风："值得注意的是，该成分可能引起部分人群过敏"
人写："对烟酰胺不耐受的姐妹绕道"
```

### 3.2 排版格式

- 每段不超过3行
- 重点内容加粗
- 用emoji分隔段落
- 图片穿插在文字之间
- 结尾引导互动

### 3.3 配图/封面要求

**封面图规范**：

| 要素 | 要求 |
|------|------|
| 比例 | 3:4（最佳） |
| 尺寸 | 1080×1440 |
| 文字 | 大字可见，3个以内核心词 |
| 风格 | 高对比、真实现场感 |
| 人物 | 优先真人出镜，避免过度修图 |

**内容配图**：
- 数量：3-9张
- 真人使用场景 > 纯产品图
- 前后对比图 > 摆拍图

## 四、话题标签使用

### 4.1 标签策略

**数量**：3-8个
**组合**：
- 1个大流量话题（如 #穿搭）
- 2-3个精准长尾话题（如 #小个子穿搭 #显高）
- 1个时效话题（如 #618必买）

### 4.2 高效标签格式

```
#种草 #好物分享 #平价护肤 #油皮亲妈 #学生党必看
```

**禁用**：
- 与内容无关的蹭流量标签
- 超过10个标签
- 重复相似标签

## 五、避坑清单

| 雷区 | 后果 | 正确做法 |
|------|------|----------|
| 极限词（最、第一） | 限流/隐藏 | 用具体数据替代 |
| 大段文字无分段 | 阅读疲劳 | 每段≤3行，emoji分隔 |
| 模糊标题 | 无点击 | 利益点+场景+数字 |
| 配图无关/模糊 | 降权重 | 高清、3:4比例、贴合内容 |
| 堆砌标签 | 限流 | 3-8个精准标签 |
| AI批量生产 | 封号 | 真人参与、增加个性化 |
| 标题党 | 举报+限流 | 标题与内容一致 |
