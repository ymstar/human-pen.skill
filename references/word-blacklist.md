## ⚠️ 绝对禁止的句式（黑名单）

```
❌ 唯有通过技术创新，才能实现可持续发展
❌ 唯有加强合作，才能应对挑战
❌ 唯有深化改革，才能解决问题

✅ 只能靠技术创新
✅ 得加强合作
✅ 得深化改革
```

---


## 🚫 替换规则

### 不是换词，是换说话方式

**错误示范**：
```
原文："人工智能技术具有重要意义"
错误改法："人工智能技术引人注目"（还是AI味）
正确改法："AI这玩意儿确实好用"
```

**错误示范**：
```
原文："首先，在工作领域..."
错误改法："在工作场景中..."（还是AI味）
正确改法："就说工作吧，"或者直接不要这个开头
```

## 🔍 黑名单检测

在改写完成后，必须检查：

1. **词汇检测**：
   - 是否使用了成语列表中的词
   - 是否使用了书面词列表中的词
   - 是否使用了套话列表中的词
   - 是否使用了假大空列表中的词

2. **句式检测**：
   - 是否使用了"唯有...才能..."
   - 是否使用了"不仅...更..."
   - 是否使用了"既要...也要..."
   - 是否使用了"随着...的发展..."
   - 是否使用了"在...领域/方面..."

3. **说话方式检测**：
   - 是否是换词，而不是换说话方式
   - 是否还是书面表达，而不是口语化

---

## ⚠️ 最高优先级

**改写禁令是最高优先级的限制**，必须严格遵守：

1. 在改写时，绝对不能使用黑名单中的词汇
2. 在改写时，绝对不能使用黑名单中的句式
3. 改写不是换词，是换说话方式
4. 改写后要通过"微信聊天测试"

**违反改写禁令的改写，即使其他方面做得再好，也是失败的。**
