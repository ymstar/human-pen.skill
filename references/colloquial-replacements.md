# 口语化替换词库

## 概览
本文档提供书面化表达到口语化表达的替换词库，用于大幅降低AI味。

## 核心替换规则

### 连接词类（必须删除或替换）

**必须删除的AI连接词**：
```
首先、其次、再次、最后、最后、另外
此外、再者、与此同时
综上所述、总而言之、总之、总的来说
值得注意的是、值得一提的是
不可否认、显然、显而易见
```

**可替换的过渡表达**：
```
原文 → 替换后
"首先" → 直接开始说，不用连接词
"其次" → "还有一点"、"还有就是"
"此外" → "对了"、"顺带说下"
"综上所述" → "总之"、"所以说"、"这么一来"
"值得注意的是" → "说个有意思的"、"关键在于"
"不可否认" → "确实"、"说实话"
```

## 使用指南

### 替换优先级
1. **最高优先级**：删除"首先、其次、此外、综上所述"等AI连接词
2. **高优先级**：替换"旨在、致力于、具有重要意义"等书面动词
3. **中优先级**：替换形容词、副词的书面表达
4. **低优先级**：根据语境灵活调整

### 场景适配
- **学术场景**：中度替换，保持专业术语，但连接词必须删除
- **自媒体场景**：重度替换，全面口语化
- **职场场景**：中重度替换，专业但自然

### 注意事项
- 不要过度口语化导致失去专业性（学术场景）
- 保持原文核心信息不变
- 替换后要通顺自然
